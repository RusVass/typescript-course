# TypeScript Курс

Этот репозиторий содержит примеры кода для курса по TypeScript.

1. **Introduction & Setup**
   - Почему TypeScript?
   - Установка и конфигурация
   - Первый TypeScript файл
   - Настройка tsconfig.json

2. **Basic Types**
   - Примитивные типы
   - Массивы и кортежи
   - Перечисления
   - Any и Unknown
   - Приведение типов

3. **Functions & Interfaces**
   - Типизация функций
   - Параметры и возвращаемые значения
   - Опциональные параметры
   - Интерфейсы vs Типы
   - Перегрузка функций

4. **Classes & Object-Oriented Patterns**
   - Классы в TypeScript
   - Свойства и методы
   - Модификаторы доступа
   - Абстрактные классы

5. **Advanced Types**
   - Объединенные и пересекающиеся типы
   - Литеральные типы
   - Mapped Types
   - Условные типы
   - Утилитарные типы

6. **Generics**
   - Обобщенные функции и классы
   - Ограничения и типы по умолчанию
   - Обобщенные функции высшего порядка

7. **Working with the DOM & Browser APIs**
   - Типы DOM
   - Обработка событий
   - HTTP клиенты

8. **Namespaces & Modules**
   - ES Modules vs Namespaces
   - Синтаксис импорта/экспорта
   - Объявления окружения

9. **Advanced Topics & Future Trends**
   - Декораторы
   - Вывод типов
   - API компилятора

10. **Professional Practices & Best Patterns**
    - Эффективные паттерны TypeScript
    - Организация кода
    - Оптимизация производительности

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите примеры:
```bash
npm start
```

3. Скомпилируйте TypeScript в JavaScript:
```bash
npm run build
```

4. Запустите в режиме наблюдения:
```bash
npm run watch
```
