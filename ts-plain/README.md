# Курс TypeScript

Цей репозиторій містить приклади коду для курсу з TypeScript.

## Структура курсу

1. **Introduction & Setup**
   - Чому TypeScript?
   - Встановлення та конфігурація
   - Перший файл TypeScript
   - Налаштування tsconfig.json

2. **Basic Types**
   - Примітивні типи
   - Масиви та кортежі
   - Переліки
   - Any та Unknown
   - Наведення типів

3. **Functions & Interfaces**
   - Типізація функцій
   - Параметри та значення, що повертаються
   - Необов’язкові параметри
   - Інтерфейси проти типів
   - Перевантаження функцій

4. **Classes & Object-Oriented Patterns**
   - Класи в TypeScript
   - Властивості та методи
   - Модифікатори доступу
   - Абстрактні класи

5. **Advanced Types**
   - Об’єднані та перехресні типи
   - Літерні типи
   - Mapped Types
   - Умовні типи
   - Утилітарні типи

6. **Generics**
   - Узагальнені функції та класи
   - Обмеження й типи за замовчуванням
   - Узагальнені функції вищого порядку

7. **Working with the DOM & Browser APIs**
   - Типи DOM
   - Оброблення подій
   - HTTP‑клієнти

8. **Namespaces & Modules**
   - ES Modules vs Namespaces
   - Синтаксис імпорту/експорту
   - Оголошення оточення

9. **Advanced Topics & Future Trends**
   - Декоратори
   - Виведення типів
   - API компілятора

10. **Professional Practices & Best Patterns**
    - Ефективні патерни TypeScript
    - Організація коду
    - Оптимізація продуктивності

## Встановлення та запуск

1. Встановіть залежності:
```bash
npm install
```

2. Запустіть приклади:
```bash
npm start
```

3. Скомпілюйте TypeScript у JavaScript:
```bash
npm run build
```

4. Запустіть у режимі спостереження:
```bash
npm run watch
```
