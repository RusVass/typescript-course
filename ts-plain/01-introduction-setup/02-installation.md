# Установка TypeScript в проект

## Глобальная установка TypeScript

Глобальная установка позволяет использовать TypeScript в любом проекте на вашем компьютере. Это удобно, если вы часто работаете с TypeScript и хотите иметь доступ к его инструментам из командной строки в любом месте.

### Установка глобально

```bash
npm install -g typescript
```

После установки вы сможете использовать команду `tsc` для компиляции TypeScript-файлов из любого места.

**Преимущества:**
- Удобство использования в разных проектах.
- Не нужно устанавливать TypeScript в каждом проекте отдельно.

**Недостатки:**
- Возможны конфликты версий, если разные проекты требуют разные версии TypeScript.

## Локальная установка TypeScript

Локальная установка TypeScript в проекте позволяет управлять версией TypeScript, используемой в конкретном проекте. Это особенно полезно, если ваш проект требует определенной версии TypeScript или если вы работаете в команде, где важно, чтобы все использовали одну и ту же версию.

### Установка локально

```bash
npm install --save-dev typescript
```

После установки вы можете использовать команду `npx tsc` для компиляции TypeScript-файлов в контексте вашего проекта.

**Преимущества:**
- Управление версиями TypeScript для каждого проекта.
- Избежание конфликтов версий между проектами.
- Легче поддерживать согласованность в команде.

**Недостатки:**
- Необходимо устанавливать TypeScript в каждом проекте отдельно.

## Рекомендации

Выбор между глобальной и локальной установкой зависит от ваших потребностей и предпочтений. В большинстве случаев рекомендуется использовать локальную установку, чтобы избежать проблем с совместимостью версий и обеспечить согласованность в команде.


## Что такое tsconfig.json и как его сгенерировать

`tsconfig.json` — это файл конфигурации для компилятора TypeScript. Он определяет корневые файлы и параметры компиляции, которые необходимы для проекта. Использование `tsconfig.json` позволяет вам настроить поведение компилятора TypeScript в соответствии с вашими потребностями.

### Генерация tsconfig.json

Чтобы сгенерировать файл `tsconfig.json`, выполните следующую команду в корне вашего проекта: `npx tsc --init`
